    MatchFetcher v0.1.0 MatchFetcher.Provider.Action View Source

    The using for the wrapping HTTP-requests.

    Example

    You can create the wrapper around SomeAction service. It returns raw HTTP-request result.

    defmodule use MatchFetcher.Provider.Actions.SomeAction do
  use MatchFetcher.Provider.Action

  def fetch(params) do
    @http_client.request(url)
  end
end
